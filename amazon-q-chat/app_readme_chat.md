## 프로젝트 개요

**사주로 해결하는 고민상담 서비스**
- 사용자의 생년월일 등 기본 정보와 질문을 입력받아 사주팔자 기반의 맞춤형 상담 제공
- 기존 GPT나 만세력 앱과의 차별화된 해석 서비스

## 핵심 차별점

- **간편한 입력**: 복잡한 만세력 캡처 과정 생략
- **대화형 상담**: 질문의 variation을 높여 디테일한 조언 제공
- **전문성**: 동양철학 기반의 정제된 해석으로 신뢰도 향상
- **접근성**: 일반인도 이해하기 쉬운 수준으로 정보 제공

## 서비스 구성

### Input
- 생년월일, 시간, 양력/음력, 성별, 이름
- **구체적인 질문**

### Output  
- **사주팔자 + 대운 + 세운** 분석
- 질문에 대한 맞춤형 해석 및 조언

### 참조 데이터
- 음양오행 (목→화→토→금→수)
- 사주간지의 십신(십성/육친) 
- 12운성, 12신살 및 기타 신살
- **대운/세운/월운 분석**

## 기술 아키텍처

### MVP 목표
- **1차**: Agent 기반 구성으로 MVP 도출
- **확장**: Multi-agent + Bedrock 구성 검토

### 시스템 구조
```
User → LLM/Agent → Response
```

### Agent 분리 구조
1. **사주팔자 파악**: 개인정보 기반 사주 계산
2. **질문 파악**: 사용자 질문 분석  
3. **응답**: 종합적인 상담 결과 제공

## UI 요구사항
- 로그인 불필요
- 핵심 기능 중심의 간단한 인터페이스
- 개인정보 입력 폼
- 질문 입력 및 답변 화면

## 향후 검토사항
- 만세력 API 활용 가능성
- Bedrock Agent 구성 방안
- 담당 SA와의 기술 검토
